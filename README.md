## 학습 방법

이 저장소는 다음과 같은 원칙으로 관리됩니다:

1. **지속적인 학습**: 매일 새로운 내용을 학습하고 기록합니다.
2. **구조화된 정리**: 주제별로 디렉토리를 구분하여 체계적으로 정리합니다.
3. **명확한 문서화**: 마크다운 형식으로 읽기 쉽게 작성합니다.
4. **참조와 인용**: 학습 출처를 명확하게 기록합니다.
5. **개인적 통찰**: 단순 정리가 아닌 자신의 생각과 경험을 함께 기록합니다.
